# apps/admin – Setup

This folder will contain the Next.js admin panel for managing content.

## Initialize locally

Run this command from the repo root:

```bash
npx create-next-app@latest apps/admin --ts --eslint --tailwind --app --src-dir --import-alias "@/*"
```

Then start the dev server:

```bash
cd apps/admin
npm run dev
```

Opens at http://localhost:3000
